# Задание 4 Задача 2

[Используя решение задачи 2](https://github.com/capti/dotNet2). Необходимо с помощью рефлексии реализовать возможность создания  и вызова методов данных классов на форме. В программе необходимо предусмотреть ввод пути к библиотеке классов, программа должна загрузить её, найти все классы, которые реализуют нужный интерфейс.  Получить всю информацию о данных классах, вывести список названий классов. При выборе класса на форму должны динамически подгружаться все методы класса с возможностью ввода параметров пользователем. При нажатии кнопки «Выполнить» должен создаваться объект и выполняться выбранный метод.
